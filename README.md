# Bien

Dự án về biển Hải Sản - Class 1 Group 7 Term 126

## Giới thiệu

Dự án nghiên cứu về biển và hải sản.

## Các file

- `index.html` - Trang web chính
- `index_new.html` - Phiên bản mới của trang web
- `convert_mhtml.js` - Script chuyển đổi MHTML
- `extract_images*.py` - Các script trích xuất hình ảnh

## Sử dụng

Mở file `index.html` hoặc `index_new.html` trong trình duyệt để xem nội dung.
